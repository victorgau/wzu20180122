# Python 研習

在本次的研習中，我們會手把手帶各位同學用 Python 做圖形處理、畫地圖、寫簡單的爬蟲、畫文字雲、並做出簡單的網頁。

課程中，軟體的部分預計會使用到 Anaconda (Python 懶人包) 及 Visual Studio Code (編輯器)，也會教同學們如何安裝。
